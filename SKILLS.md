 # 프로젝트 개발 환경

 1. 프로젝트 개요
   * 프로젝트 명칭: 만 13세 미만 아이들을 위한 OTT '루키즈'
   * 프로젝트 기간: 2026.03.24(화) ~ 2026.04.03(금) (주말 제외 총 9일)

  2. 팀 구성 및 역량
   * 팀 인원: 총 3인
       * A: 디자인 리드
       * B: 프론트엔드 리드
       * C: 백엔드/AI 리드
   * 팀 숙련도: 디자인 80% / 코딩 20% (코드는 Gemini CLI를 활용한 '바이브 코딩' 방식으로 구현)

  3. 기술 스택
   * 기획 및 디자인: Google Stitch (아이데이션), Figma (디자인 시스템)
   * 프론트엔드: React 19, Tailwind CSS v4, react-player, React Router v7, Axios, FontAwesome, tailwind-merge       
   * 백엔드 및 AI: Python FastAPI, HuggingFace Qwen2.5-72B-Instruct, 생성형 AI 비디오, TMDB API
   * 도구 및 배포: Gemini CLI (Git + 바이브 코딩), Render (배포)

  4. 수행 프로세스 (총 9일)
   1. 기획 (2일): 서비스 방향성 및 요구사항 정의
   2. 디자인 (3일): UI/UX 디자인 및 디자인 시스템 구축
   3. 구현 (3일): 프론트엔드 및 백엔드 개발 (Gemini CLI 활용)
   4. 검토 및 배포 (1일): 최종 테스트 및 Render를 통한 배포

  5. 배포 및 프로젝트 구조
   * 구조: 모노레포(Monorepo) 방식 (frontend/ + backend/)
   * 배포 환경: Render

  6. 중요사항
   * 주 타겟층 : 키즈(만 4-7세), 주니어(만 8-12세)
   * AI 가이드 '루'가 친절하고 활기차게 아이들에게 맞춤 콘텐츠를 추천
